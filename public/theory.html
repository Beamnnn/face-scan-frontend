WORK DETAIL

1. การทำงานภายใน MongoDB

     ระบบจดจำใบหน้าใช้ MongoDB ในการจัดเก็บข้อมูลใบหน้าที่ลงทะเบียนไว้ โดยมีโครงสร้างฐานข้อมูลดังนี้:

  • โครงสร้างฐานข้อมูล - ข้อมูลถูกเก็บใน Collection ชื่อ registered_faces โดยแต่ละเอกสารมีฟิลด์สำคัญ ได้แก่:
    • _id - รหัสเอกสารที่ MongoDB กำหนดอัตโนมัติ
    • name - ชื่อของบุคคลที่ลงทะเบียน
    • encoding - อาร์เรย์ค่าตัวเลข 128 ค่า ที่เป็นตัวแทนใบหน้า
  • การเพิ่มข้อมูล (Insert Data) - เมื่อผู้ใช้ลงทะเบียน ระบบจะสร้าง Face Embedding และบันทึกลง MongoDB
  • การค้นหาข้อมูล (Query Data) - ระบบใช้ Vector Matching เทียบค่า encoding กับฐานข้อมูลเพื่อตรวจสอบใบหน้าที่ตรงกัน
  • การอัปเดตและลบข้อมูล (Update & Delete) - สามารถอัปเดตหรือลบข้อมูลบุคคลที่ลงทะเบียนได้

2. Machine Learning Model

      ใช้้โมเดล Machine Learning ที่ใช้ในโปรเจคนี้คือ Random Forest โดยใช้ RandomForestClassifier จาก sklearn (scikit-learn) ซึ่งเป็นโมเดลที่สามารถเรียนรู้และทำนายข้อมูลที่มีความซับซ้อนได้ดี ในกรณีของโปรเจคนี้ โมเดล Random Forest ใช้ feature vectors ที่ได้จากการแปลงภาพใบหน้าเป็นข้อมูลตัวเลข (ผ่านการแปลงภาพ เช่น การใช้ face_encodings จาก face_recognition library) เพื่อทำการฝึกและทำนายว่าผู้ใช้คือใคร โมเดลนี้ใช้เพื่อการ จำแนกประเภทใบหน้า ที่เก็บไว้ในฐานข้อมูล เช่น เมื่อผู้ใช้สมัครใหม่ ระบบจะเก็บข้อมูลใบหน้าและเมื่อมีการสแกนใบหน้าใหม่ จะใช้โมเดลนี้ในการทำนายว่าผู้ใช้คนนั้นคือใครเพื่อช่วยจำแนกใบหน้า โดยใช้อัลกอริทึม เช่น:

3. Neural Network Model

      ใช้ โมเดล Neural Network (CNN) จะใช้สำหรับการ จำแนกใบหน้า โดยใช้ Convolutional Neural Networks (CNN) ซึ่งเป็นเทคนิคที่เหมาะสำหรับงานที่เกี่ยวข้องกับ การรู้จำภาพ เช่น Face Recognition โมเดล CNN จะเรียนรู้คุณลักษณะต่าง ๆ ของใบหน้า เช่น รูปร่างของตา จมูก และปาก เพื่อสามารถจำแนกได้ว่าใครเป็นเจ้าของใบหน้านั้น CNN มักจะใช้ในการจำแนกภาพที่มีความซับซ้อนสูง ซึ่งในกรณีนี้จะมีความแม่นยำสูงกว่า Random Forest เมื่อใช้กับข้อมูลภาพ ในโปรเจคนี้ที่คุณกำลังพัฒนา:

  • Random Forest ใช้สำหรับการ จำแนกใบหน้า จากฟีเจอร์ที่ได้จากการแปลงภาพเป็นข้อมูลตัวเลข
  • Neural Network (CNN) จะใช้สำหรับ การจำแนกใบหน้า จากภาพโดยตรง โดยเรียนรู้ลักษณะต่าง ๆ ของใบหน้า

4. การทำงานของระบบจดจำใบหน้า

  • การลงทะเบียนใบหน้า (Register Face) - ผู้ใช้ต้องอัปโหลดภาพใบหน้าและระบบจะบันทึกข้อมูลลงฐานข้อมูล
  • การสแกนใบหน้า (Scan Face) - ระบบจะนำภาพที่อัปโหลดมาเปรียบเทียบกับฐานข้อมูลเพื่อตรวจสอบว่าใบหน้านั้นตรงกับบุคคลใด
